# GOGO BANG ( 교육용 창의력 탈출게임 )
## Introduction
아이템을 적재적소에 사용해가며 방을 탈출하는 형식에 교육적, 창의적 요소를 더한 게임


## Environment / Requirement
Unity2018  
Windows10  
모든 C# 코드는 Visual Studio2017 에서 작성


## Goal
현재 완성된 하나의 방(Stage)를 탈출하기 위해선 컴퓨터지식을 알아야함.  
추가로 개발을 한다면 Stage마다 요구하는 지식을 다르게 하여, 여러버전 제작하는 것을 목표로 둠


## Project composition
#### 주요 클래스
다이얼로그 제어  
DialogueManager.cs / Dialogue.cs / DialogueTrigger.cs

이벤트 제어
